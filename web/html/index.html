Présentation du sujet:

Dans le cadre de la première année de BUT Informatique, nous avons du réaliser un projet mettant en relation les notions obtenues en :

  • Communication
  • Programmation
  • Web

Parmis les différents sujets proposés, j'ai choisi de traiter le sujet lié aux discours de nouvel an de chaque président de la 5ème République:

Dire quels sont les thèmes majeurs, quelles sont les idées dominantes, dans les messages de vœux de Nouvel An adressés aux Français par les présidents de la République chaque année. A cette fin, on produira un nuage de mots pour l’ensemble. Mais, pour les étudiants volontaires, il est également possible d’introduire un paramètre de temps : un nuage de mots par période (découpage libre), ou un seul nuage de mots qui se modifie automatiquement pour refléter l’évolution temporelle.
